# Janajātiya Oral Traditions and the Animistic Roots of Sanātana Dharma

## Article Structure

This folder contains the multilingual content for the research article exploring the animistic and tribal roots of Sanātana Dharma, pairing oral traditions with archaeological, geological, and epigraphic evidence.

### Content Files

- **`en.md`**: English source content (canonical version, ~1,100 lines after bibliography cleanup)
- Future translations will be added as `hi.md`, `ta.md`, `bn.md`, etc.

### Translation Directory

Translation metadata and structured content live under:
- `i18n/articles/janajati-oral-traditions/<lang>.json`

### Key Features

- **Evidence-graded methodology**: Each major claim receives a confidence grade (A–D)
  - **A**: Primary material evidence (excavations, stratigraphy, scientific data)
  - **B**: Multiple convergent secondary sources
  - **C**: Inference with partial support
  - **D**: Oral memory or textual testimony without material confirmation
  
- **16 academic citations**: Peer-reviewed sources in clean MLA 9 format
- **3 figures**: Timeline panel, ritual comparanda, heritage integration schematic
- **8 major sections**: From Harappan fire-altars to Jagannātha tribal cults
- **Pan-Indian scope**: Sites from Rajasthan to Odisha to Andaman Islands

### Academic Anchors (A-Grade Evidence)

1. **Kalibangan fire-altars** (Harappan-Vedic continuity)
2. **Hastināpura flood stratum** (epic archaeology)
3. **Ghaggar-Hakra palaeochannel** (Sarasvatī river)
4. **Dvārakā submerged architecture** (marine archaeology)
5. **Jagannātha Daitā priesthood** (living tribal-Sanskritic synthesis)

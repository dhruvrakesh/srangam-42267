   - Conservative: Max Müller, modern cautious (~1500 BCE)

   - Minimum date: 3rd millennium BCE (high confidence)

## Evidence Grading System

**A-Grade (High Confidence):**
- Sarasvatī River drying ~1900 BCE (geological)
- Mitanni treaty 1380 BCE (archaeological)

**B-Grade (Strong Support):**
- B.B. Lal 3rd millennium synthesis (multiple sources)
- Puranic chronology 3102 BCE (convergent texts)

**C-Grade (Moderate Inference):**
- Tilak/Jacobi astronomy ~4500 BCE (requires interpretation)
- Talageri internal chronology (single-source inference)

**D-Grade (Speculative):**
- Saroj Bala 6000-8000 BCE (limited acceptance)
- Vedveer Arya 14500 BCE (highly contentious)

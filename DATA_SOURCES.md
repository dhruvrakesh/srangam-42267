### Confidence Level System

#### Source Reliability Categories

**Category A: Highest Confidence (90-100%)**
- Contemporary inscriptions with secure archaeological context
- Dated archaeological materials with stratigraphic provenance
- Multiple independent source confirmation

**Category B: High Confidence (75-89%)**
- Archaeological evidence with relative dating
- Textual sources with historical corroboration
- Consistent pattern across multiple sites

**Category C: Moderate Confidence (60-74%)**
- Single source with probable accuracy
- Archaeological evidence with dating uncertainties
- Textual sources requiring interpretation

**Category D: Approximate (45-59%)**
- Reconstructed locations based on textual descriptions
- Archaeological evidence with dating controversies
- Sources requiring significant scholarly interpretation

### MLA Citation Standards

#### Primary Sources
```
Ancient Author. "Text Title." In Collection Name, edited by Editor Name, pages. Publisher, Year.

Example:
Valmiki. "Kishkindha Kanda." In The Ramayana, translated by Ramesh Menon, 245-267. North Point Press, 2001.
```

#### Archaeological Reports  
```
Lead Archaeologist. "Site Name: Excavation Report." Journal Name vol. number (Year): pages.

Example:
Cherian, P.J. "Pattanam Excavations 2007-2012: New Evidence for Ancient Muziris." Journal of Indian Ocean Archaeology 8 (2015): 1-25.
```

#### Inscriptional Evidence
```
"Inscription Name." In Corpus of Inscriptions, edited by Editor, inscription number. Institution, Year.

Example:
"Kutai Yupa Inscription." In Corpus Inscriptionum Javanicarum, edited by J.G. de Casparis, K.1. Royal Netherlands Academy, 1975.
```
